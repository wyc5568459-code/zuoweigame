## 游戏 2：佐为自走棋

`games/game2/` 是一个纯 HTML/CSS/JavaScript 的单机自走棋网页游戏（**手机端优先适配**，桌面端也可游玩），包含以下核心玩法：

- 8+ 棋子、费用/星级/阵营/职业属性
- 3 合 1 升星（1星→2星、2星→3星）
- 经济系统：基础收入、利息、刷新商店、买经验升级
- 4x4 棋盘 + 8 格备战区，支持点击选中与放置
- 人口上限（由等级决定）
- 羁绊系统（战士/猎手/法师/山林，多档激活）
- 自动战斗回合（准备阶段 ↔ 战斗阶段）
- localStorage 保存当前局面、最高回合与历史战绩

### 佐为自走棋操作说明

1. **买棋子**：点击底部商店卡片，消耗对应金币，棋子进入备战区（手机端可左右滑动商店）。
2. **摆阵上场**：先点击棋子，再点击目标格；可在棋盘和备战区之间互换。
3. **刷新商店**：点击“刷新商店 (2金)”。
4. **升级人口**：点击“购买经验 (4金)”，经验满后自动升级并提高人口上限。
5. **开始战斗**：点击“开始战斗”，系统会生成敌方阵容并自动对战。
6. **结算与继续**：胜利得金币，失败扣血，回合自动推进。
7. **重新开局**：点击“重新开局”清空当前对局数据。

## 游戏 3：佐为のタワー（魔塔50层）

`games/game3/` 为手机优先的单机网页 RPG，支持 1F~50F 完整流程：

- 目标：49F 救出公主，50F 击败魔王通关
- 核心系统：黄/蓝/红钥匙与门、宝石加攻防、血瓶、金币商店、陷阱扣血
- 战斗：接触怪物前可看到预计损耗 HP，内置怪物图鉴（菜单中查看）
- 操作：屏幕方向键 + 确认/取消/菜单，桌面支持键盘方向键
- 存档：localStorage 三存档位，支持随时保存与读取

### 佐为のタワー操作说明

1. 点击虚拟方向键或使用键盘方向键移动。
2. 走到怪物旁会显示预计损耗 HP，撞上怪物即战斗。
3. 点击“菜单”可打开怪物手册、存档/读档与剧情目标。
4. 商店楼层可消耗金币提升 HP/ATK/DEF 或购买红钥匙。
5. 49F 触发救公主剧情后，前往 50F 击败魔王。

## 启用 GitHub Pages

1. 进入仓库 `Settings` -> `Pages`
2. `Build and deployment` 中选择：
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. 保存后等待 GitHub 完成发布
