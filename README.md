# zuoweigame（左为游戏）

一个可部署到 GitHub Pages 的纯前端小游戏仓库。

## 项目结构

```text
.
├── index.html              # 游戏总入口页
└── games/
    ├── game1/
    │   ├── index.html      # 游戏 1：反应速度挑战
    │   ├── style.css
    │   └── script.js
    └── game4/
        ├── index.html      # 游戏 4：佐为自走棋
        ├── style.css
        └── script.js
```

## 新增游戏：佐为自走棋

`games/game4/` 是一个纯 HTML/CSS/JavaScript 的单机自走棋网页游戏（**手机端优先适配**，桌面端也可游玩），包含以下核心玩法：

- 8+ 棋子、费用/星级/阵营/职业属性
- 3 合 1 升星（1星→2星、2星→3星）
- 经济系统：基础收入、利息、刷新商店、买经验升级
- 4x4 棋盘 + 8 格备战区，支持点击选中与放置
- 人口上限（由等级决定）
- 羁绊系统（战士/猎手/法师/山林，多档激活）
- 自动战斗回合（准备阶段 ↔ 战斗阶段）
- localStorage 保存当前局面、最高回合与历史战绩

### 佐为自走棋操作说明

1. **买棋子**：点击底部商店卡片，消耗对应金币，棋子进入备战区（手机端可左右滑动商店）。
2. **摆阵上场**：先点击棋子，再点击目标格；可在棋盘和备战区之间互换。
3. **刷新商店**：点击“刷新商店 (2金)”。
4. **升级人口**：点击“购买经验 (4金)”，经验满后自动升级并提高人口上限。
5. **开始战斗**：点击“开始战斗”，系统会生成敌方阵容并自动对战。
6. **结算与继续**：胜利得金币，失败扣血，回合自动推进。
7. **重新开局**：点击“重新开局”清空当前对局数据。

## 本地游玩

无需安装依赖，直接在浏览器中打开以下文件即可：

- `index.html`：查看游戏列表
- `games/game1/index.html`：直接进入反应速度挑战
- `games/game4/index.html`：直接进入佐为自走棋

## 启用 GitHub Pages

1. 进入仓库 `Settings` -> `Pages`
2. `Build and deployment` 中选择：
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. 保存后等待 GitHub 完成发布

## 访问路径示例

发布后可通过类似路径访问：

- 首页：`/`
- 游戏 1：`/games/game1/`
- 游戏 4（佐为自走棋）：`/games/game4/`
